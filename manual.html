Das Internet

Was ist eigentlich das Internet?

Wenn man heute allgemein über das Internet spricht, dann meinen die meisten Menschen damit eigentlich das World Wide Web, abgekürzt WWW. Tatsächlich aber muss man diese Begriffe voneinander trennen:

Das Internet ist genau gesehen eine Vernetzung von wirklich sehr vielen Computern, bestehend aus Computern und Kabeln usw. und Regeln, wie die einzelnen Computer miteinander reden, die Protokolle genannt werden.

Das WWW hingegen baut auf dem Internet auf und beschreibt ein Konzept, das beschreibt wie man Inhalte miteinander teilt. Dazu folgt später noch mehr.

  • Das Internet ist ein Weltweites Computernetzwerk
  • Das World Wide Web (WWW) ist die bekannteste Art der Nutzung des Internets

Wie alles entstanden ist

Alles begann 1969 mit der Vernetzung von 4 Forschungsinstituten und Universitäten in den USA.
Diesem Netzwerk, das ARPANET genannt wurde, schlossen sich im Laufe der Jahre immer mehr Einrichtungen, so dass im Jahr 1983 bereits 68 Knotenpunkte miteinander verbunden waren.

1989 war das ARPANET durch die Vielzahl der Nutzer an seine Grenzen geraten und ein neues Netz, das heutige Internet wurde geschaffen. Dieses Netz behält die Grundtechnologien des APRANETs bei, entwickelt sie aber auch weiter, damit mehr und mehr neue Knoten angeschlossen werden können.

  • Am Anfang waren lediglich Universitäten und Forschungseinrichtungen in USA angeschloßen (ARPANET)
  • Immer mehr Einrichtungen auch außerhalb der USA kamen hinzu
  • 1990 wird das ARPANET durch das heutige Internet ersetzt

Die Vernetzung von Computern

Man kann sich jeden Computer als einen Knoten vorstellen. Von jedem Knoten gehen ein oder mehrere Fäden zu anderen Knoten. Heute gibt es so viele Knoten im Internet, dass man sie eigentlich gar nich mehr darstellen kann. Wenn man einen Versuch startet, würde es in etwa so aussehen: Klick mich!

Die Art und Weise der Vernetzung von Computern wirkt auf den ersten Blick etwas chaotisch, denn es gibt zwar große und kleine Knoten, aber kein wirkliches Zentrum. Aber das ist so gewollt, denn es hat den Vorteil, das falls einer der Knoten ausfällt, einfach ein anderer seine Aufgaben übernehmen kann.

Wie viele Knoten es gibt ist aber eigentlich gar nicht so wichtig. Das entscheidende ist, dass es von jedem einzelnen irgendeinen Weg zu jedem anderen Knoten gibt.

  • Computernetze sind dezentral aufgebaut
  • Fällt einer aus, kann ein anderer Einspringen
  • Jeder Knoten kann von überall erreicht werden

Das World Wide Web (WWW)

1991 wurde das WWW in seiner heutigen Form von Tim Berners Lee veröffentlicht. Es besteht in dieser Form bis heute und jeder der das Internet nutzt kennt dieses Konzept, obwohl die meisten von uns es gar nicht wissen. Ein Beispiel:

Ich bin ein Link und der Ort auf den ich zeige wird URL genannt. Du kannst ihn sehen, wenn du mich anklickst!

Durch die Entwicklung dieser Struktur und wegen der ersten grafikfähigen Browser wurde das Internet auch für Laien nutzbar, denn jetzt konnte man echte Texte und Bilder einfach darstellen. Diese Darstellung übernimmt der Browser.

  • Das WWW wurde 1991 von Tim Berners Lee vorgestellt
  • Die Struktur des WWW ist bis heute die Grundlage aller Internetseiten
  • Durch das WWW und durch die Browser wurde das Internet für jeden nutzbar

Webserver und Browser

Ein Browser ist ein Programm, das die Informationen aus dem Internet für Menschen besser lesbar gestaltet. Du benutzt gerade einen Browser, um diese Seite zu lesen. Vielleicht hast du auch schon einmal von Firefox, Internet Explorer, Chrome oder Safari gehört?

Der Browser erhält die Informationen in dem er eine Anfrage an einen Webserver stellt. Der Webserver antwortet, in dem er den HTML Code zurückschickt.

Man kann diesen Vorgang mit einem Einkauf beim Bäcker vergleichen. Der Kunde betritt den Laden und zeigt auf ein Stück Brot. Er stellt dadurch eine Anfrage. Der Bäcker greift in die Auslage, holt das Brot und reicht es dem Kunden. Dadurch erhält der Kunde was er will. Das Zeigen auf das Brot ist die Anfrage. Im Internet ist das eine Eingabe in den Browser.
Danach erfolgt die übergabe des Brotes. Im Internet wäre das die übermittlung des HTML-Codes über das Netzwerk und die Darstellung im Browser.

  • Browser "übersetzten" HTML und machen es einfach lesbar
  • URLs sind eindeutige Zeiger auf Inhalte im Internet
  • Anfragen an URLs werden von dem jeweiligen Webserver beantwortet

Uniform Resource Locator (URL)

Hinter jedem Link und hinter jeder Eingabe in der Befehlszeile des Browser steckt eine URL. Eine URL ist ein Zeiger, der auf genau einen Inhalt gerichtet ist. Durch die URL kann man so eindeutig bestimmen, welchen Inhalt man gerne erreichen möchte und es kann zu keinen Verwechslungen kommen.

Eine URL sieht meistens so aus: http://www.afr.musin.de/

Der erste Teil ("http://") sagt welches Protokoll verwendet wird (HTTP). Der zweite ("www.") besagt lediglich, dass es sich um eine Anfrage an das World Wide Web handelt. Da diese Teile meistens gleich kann man sie bei einer EIngabe im Browser auch weglassen. Sie werden dann automatisch ergänzt.

Danach kommen der Name (afr.musin) und die Domain (.de/). Der Name ("afr.musin") ist zusammen mit einer Domain (".de") der eindeutige Teil, der zur Bestimmung der Ressource dient. Diese URLs werden durch bestimmte Regeln so aufgelöst, dass bekannt ist welcher Server wo auf der Welt genau diesen Inhalt, den man gerade möchte bereitstellt.

  • Eine URL ist ein eindeutiger Zeiger auf eine Ressource
  • Hinter jedem Link und jeder Eingabe im Browser steckt eine URL
  • Anfragen an URLs werden von dem jeweiligen Webserver beantwortet

Zusammenfassung

  • Das Internet ist ein ziemlich großer Haufen von kleineren und größeren Computern, der mittlerweile über die ganze Welt verteilt ist.
  • Das WWW ist die populärste Art und Weise, wie wir das Internet nutzen können.
  • Die Inhalte des WWW werden auf Webservern bereitgestellt.
  • Der Browser stellt die Inhalte des Intenets in einer gut lesbaren Form dar.
